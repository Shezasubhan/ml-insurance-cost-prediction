# Optimized Machine Learning for Insurance Cost Prediction

## Overview

This project explores the application of machine learning and explainable artificial intelligence (XAI) techniques for predicting healthcare insurance costs.

The objective was to develop accurate predictive models while identifying the key factors that influence insurance charges. Multiple machine learning algorithms were evaluated and optimized to improve predictive performance and model interpretability.

---

## Research Paper

**Optimized Machine Learning for Insurance Cost Prediction**

Preprint Publication:

https://www.preprints.org/manuscript/202502.1777

---

## Dataset

Source: Medical Insurance Cost Dataset

The dataset includes demographic and health-related information such as:

* Age
* Sex
* Body Mass Index (BMI)
* Number of Children
* Smoking Status
* Geographic Region

Target Variable:

* Individual Medical Insurance Charges

---

## Methodology

The project followed a complete machine learning workflow:

### Data Preparation

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Encoding
* Data Transformation

### Machine Learning Models

* Ridge Regression
* Random Forest Regression
* XGBoost Regression

### Model Optimization

* Optuna Hyperparameter Tuning
* Cross-Validation
* Performance Comparison

### Explainability

* SHAP (SHapley Additive Explanations)
* Feature Importance Analysis
* Model Interpretation

---

## Results

XGBoost achieved the strongest predictive performance among the evaluated models.

### Key Cost Drivers Identified

* Smoking Status
* Body Mass Index (BMI)
* Age

SHAP analysis provided interpretable explanations for model predictions and highlighted the relative influence of each feature on insurance costs.
